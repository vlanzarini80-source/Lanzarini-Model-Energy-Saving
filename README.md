## 📌 Descrizione del Progetto
Il **Modello Lanzarini** introduce un paradigma rivoluzionario nell'ottimizzazione delle reti neurali, finalizzato alla drastica riduzione del consumo energetico dei Data Center. A differenza degli ottimizzatori stocastici tradizionali, questo modello utilizza lo **Stato W** quantistico come guida geometrica per il movimento dei pesi.

## 🧬 Fondamento Matematico
L'innovazione principale risiede nell'integrazione di un termine di curvatura entropica basato sull'entropia di entanglement di Von Neumann. La dinamica dell'aggiornamento dei pesi segue l'equazione:

$$\frac{dw}{dt} = -(\nabla L + \alpha \nabla S_{ent})$$

Dove:
- $\nabla L$ è il gradiente della funzione di perdita (Loss).
- $S_{ent}$ è l'entropia calcolata sulle matrici densità ridotte del sistema.
- $\alpha$ è il coefficiente di curvatura geodetica.
